# MaxPooling


では最後にプーリングについて説明します。プーリング層では、畳み込み層と同じように、スライドさせたフィルターの範囲の最大値を返します。下の図のように、フィルターで囲まれた2×2の範囲で最大値を取り出します。
![alt text](<スクリーンショット 2026-04-13 175132.png>)


このプーリングが前回の畳み込み層と大きく異なる点は主に二つです。
- **重みがない**   

  プーリング層はただ範囲内の最大値を取りだすだけなので、\\(W\\)に値する重みが存在しません。よってオプティマイザーで重みを更新することなく、ただ値を計算する関数です。

- **チャンネル数が維持される**   

  プーリング層ではチャンネルは入力データと出力データで変化しません。それはチャンネルごとに独立して処理されるからです。先ほどの畳み込み層の場合は、入力データのチャンネル数が\\(C\\)だったのに対し、出力データでは\\(OC\\)とチャンネル数が変化します。



  ではこのプーリング計算を実際に行列計算として行う方法を考えます。

![alt text](<スクリーンショット 2026-04-14 180147.png>)

はじめに畳み込み層と同じようにフィルターを用いるので、Im2col関数に関しては同じです。これにより、縦の列は一つのフィルターの範囲のデータとなります。(黄色で示したところ)次に **permuted_axes** という軸を入れ替える関数を用います。この関数は、補足の[Permuted_axes]()で説明、実装します。今回の場合、\\((N,C\cdot KH\cdot KW,OH\cdot OW)\\)が、\\((N,OH\cdot OW,C\cdot KH\cdot KW)\\)というように、1番目と2番目の軸が入れ替わります。次にチャンネルごとに最大値を取るため、チャンネル数である\\(C\\)を分離させます。これにより、\\(C\cdot KH\cdot KW\\)というように一つのフィルターの内部からチャンネルごとに分離させられます。最後に行ごとに最大値を取ることで、チャンネルで独立した、フィルター内の最大値を返すことができます。あとはこの行列を\\((N,C\cdot OH\cdot OW,1)\\)と変形させれば、完了です。出力データを見るとわかるように、チャンネル数は\\(C\\)と入力の時と変化していません。
